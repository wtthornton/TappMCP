Complete Demo System Guide

The TappMCP demo system provides interactive HTML reports that showcase the platform's capabilities through real-time testing and performance measurement.

What Are the Demos?

Quick Demo

Tools Tested: Smart Plan Enhanced, Smart Write, Smart Analyze
Duration: ~5 seconds
Purpose: Quick platform overview
Script: node scripts/quick-mcp-demo.js

Comprehensive Demo

Tools Tested: All 21 tools
Duration: ~30 seconds
Purpose: Complete platform testing
Script: node scripts/comprehensive-mcp-demo.js

Technical Documentation Demo

Content: Technical documentation overview
Purpose: Documentation demonstration
Generated: Automatically with demos

Complete Documentation Demo

Content: All documentation links and overview
Purpose: Complete documentation access
Generated: Automatically with demos

How Demos Work

Execution Process

  1. Initialization: MCP server startup and tool registry loading
  2. Tool Testing: Individual tool execution with performance measurement
  3. Grading: A-F performance grading based on response times
  4. Report Generation: Interactive HTML report creation
  5. File Output: HTML report saved to file system

Performance Measurement

  • Response Time: Individual tool execution time
  • Success Rate: Percentage of successful operations
  • Grade Calculation: A+ (‚â§50% target) to F (>120% target)
  • Business Value: ROI and cost-benefit analysis

Interactive Features

  • Expandable Results: Click to view detailed tool outputs
  • Performance Charts: Visual performance metrics
  • Grade Indicators: Color-coded performance grades
  • Export Options: Download reports in various formats

Running the Demos

Quick Demo

# Run quick demo
node scripts/quick-mcp-demo.js

# Output: mcp-demo-report-[timestamp].html

Comprehensive Demo

# Run comprehensive demo
node scripts/comprehensive-mcp-demo.js

# Output: mcp-comprehensive-demo-[timestamp].html

Viewing Reports

  1. Open the generated HTML file in your browser
  2. Click on tool results to expand details
  3. View performance metrics and grades
  4. Export or share the report

Troubleshooting

Common Issues

Performance Issues

  • Symptoms: Slow response times, high memory usage
  • Solutions: Check timeout values, limit concurrent requests, verify memory usage

Test Failures

  • Symptoms: Tests failing, coverage below threshold
  • Solutions: Fix broken tests, add missing test cases, check configuration

Tool Execution Errors

  • Symptoms: Tools failing, error messages
  • Solutions: Validate input data, check schema compliance, verify external services
