## üöÄ **Getting Started**

### **Prerequisites**
- Node.js 18+
- TypeScript 5+
- SQLite 3.40+

### **Setup Commands**
```bash
# Install dependencies
npm install better-sqlite3 @types/better-sqlite3

# Create database directory
mkdir -p data/sqlite

# Run migrations
npm run db:migrate

# Start development
npm run dev
```
